reply

reply(room?: String, msg: String, hideToast?: Boolean): void | Boolean

특정 방 또는 메시지가 온 방에 메시지를 보냅니다.



파라미터

  • room? : 메시지를 보낼 방 이름
  • msg : 보낼 메시지의 내용
  • hideToast? : room?의 값이 있을 경우, 방의 세션을 가지고 있는지를 토스트 메시지로 출력할 것인지의 여부 (기본값: false)


    • 리턴 값

  • (room? 값이 있을 경우) room?방에 대한 답장 세션을 가지고 있으면 true, 아니면 false

    • replyDelayed

    replyDelayed(room?: String, msg: String, delay: Long, hideToast?: Boolean): void | Boolean

    특정 방 또는 메시지가 온 방에 지정한 시간이 지난 후 메시지를 보냅니다.



    파라미터

  • room? : 메시지를 보낼 방 이름
  • msg : 보낼 메시지의 내용
  • delay : 답장 지연 시간 (단위: 밀리초)
  • hideToast? : room?의 값이 있을 경우, 방의 세션을 가지고 있는지를 토스트 메시지로 출력할 것인지의 여부 (기본값: false)


    • 리턴 값

  • (room? 값이 있을 경우) room?방에 대한 답장 세션을 가지고 있으면 true, 아니면 false

    • markAsRead

    markAsRead(room?: String, packageName?: String): Boolean

    특정 방 또는 메시지가 온 방의 메시지를 읽음 처리합니다.



    파라미터

  • room? : 메시지를 읽음 처리할 방 이름
  • packageName? : 메시지를 읽음 처리할 메신저 앱의 패키지명


    • 리턴 값

  • 알림 읽기 액션을 찾을 수 있으면 true, 아니면 false
