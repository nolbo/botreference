response

response(room, msg, sender, isGroupChat, replier, imageDB, packageName)

카카오톡 봇 앱이 사용자가 지정한 메신저 앱의 알림을 성공적으로 파싱했을 경우 호출되는 이벤트 리스너입니다.



파라미터

  • room : 메시지가 온 채팅방의 이름
  • msg : 메시지의 내용
  • sender : 메시지를 보낸 사람의 닉네임
  • isGroupChat : 3인 이상의 채팅방 인지의 여부 (3인 이상일 경우true, 아니면 false)
  • replier : 답장을 보내기 위한 객체
  • imageDB : 프로필 사진 정보를 담고 있는 객체
  • packageName : 채팅이 온 메신저 앱의 패키지명
