message이벤트를 발생시킨 메시지에 대한 정보를 담고있는 인자입니다.



파라미터

  • room : 명령어가 수신된 방 이름
  • content : 수신된 메시지의 내용
  • isGroupChat : 방 내 인원이 3인 이상이면 true, 아니면 false
  • isDebugRoom : 디버깅룸에서 수신된 메시지면 true, 아니면 false
  • author : 메시지 전송자에 대한 정보를 담고 있는 객체
    • name : 메시지를 전송한 수신자의 닉네임
    • avatar : 수신자의 프로필에 대한 정보
      • getBase64 : 수신자 프로필의 Base64값을 반환하는 함수
      • getBitmap : 수신자 프로필에 대한 android.graphics.Bitmap객체를 반환하는 함수
  • reply : content 라는 내용의 메시지를 전송하는 함수
  • markAsRead : 메시지가 수신된 방의 알림을 읽음으로 표시하는 함수 (성공하면 true, 아니면 false 반환)
  • packageName : 메시지가 수신된 메신저 앱의 패키지명
