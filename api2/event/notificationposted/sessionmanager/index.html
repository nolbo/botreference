bindSession

bindSession(packageName?: String, room: String, action?: android.app.Notification.Action): Boolean

다른 메신저앱의 특정 방에 채팅이 수신되었을 경우, 답장을 전송합니다.



파라미터

  • packageName? : 답장을 수신할 메신저 앱의 패키지명
  • room : 답장을 수신할 메신저 앱 내의 채팅방
  • action? : 답장 기능을 수행하기 위한 android.app.Notification.Action값


    • 리턴 값

  • action?과 메신저봇이 자동 분석한 답장 액션 모두 null이면 false, 아니면 true
