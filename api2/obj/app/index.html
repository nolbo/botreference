앱의 Context를 가져옵니다.



리턴 값

  • android.content.Context값

    • runOnUiThread

    runOnUiThread(task: Function, onComplete: Function): void

    특정 함수를 UI쓰레드에서 호출합니다.



    파라미터

  • task : UI쓰레드에서 실행할 함수
  • onComplete : task 함수 실행 후 호출될 함수
    • error : task 함수 실행 도중 에러 발생 시 에러 객체를 전달받는 인자 (java.lang.Throwable)
    • result : task 함수의 반환값을 전달받는 인자
