send

send(broadcastName: String, value: Any): void

모든 스크립트를 대상으로 특정 요소를 방송합니다.



파라미터

  • broadcastName : 방송의 이름
  • value : 방송으로 전파할 값 (register()와 unregister()의 인자 task의 인자로 주어짐.)

    • register

    register(broadcastName: String, task: Function): void

    특정 브로드캐스트에 리스너를 등록합니다.



    파라미터

  • broadcastName : 리스너를 등록할 대상 브로드캐스트
  • task : broadcastName 브로드캐스트의 리스너로 등록할 함수
    • value : broadcastName 브로드캐스트의 방송으로 전달받은 값 (send()의 두 번째 인자의 값을 전달받음.)

    • unregister

    unregister(broadcastName: String, task: Function): void

    특정 리스너를 특정 브로드캐스트로부터 해제합니다.



    파라미터

  • broadcastName : 리스너를 해제할 대상 브로드캐스트
  • task : broadcastName 브로드캐스트로부터 해제할 리스너

    • unregisterAll

    unregisterAll(): void

    특정 브로드캐스트에 등록된 모든 리스너를 해제합니다.
