setCommandPrefix

setCommandPrefix(prefix: String): void

카카오톡 봇 명령어의 접두어를 설정합니다. 설정된 접두어로 시작하는 메시지를 감지할 시 Event.COMMAND를 발생시킵니다.



파라미터

  • prefix : 카카오톡 봇 명령어의 접두사로 설정할 문자열 값

    • send

    send(room: String, msg: String, packageName?: String): Boolean

    특정 방에 메시지를 보냅니다.



    파라미터

  • room : 메시지를 보낼 방 이름
  • msg : 보낼 메시지의 내용
  • packageName? : 메시지를 보낼 메신저 앱의 패키지명


    • 리턴 값

  • room 방에 대한 세션이 존재하면 true, 아니면 false

    • canReply

    canReply(room: String, packageName?: String): Boolean

    특정 방에 대한 세션을 가지고 있는지의 여부를 반환합니다.



    파라미터

  • room : 세션 존재 여부를 확인할 방의 이름
  • packageName? : 세션 존재 여부를 확인할 방이 소속된 메신저 앱의 패키지명


    • 리턴 값

  • room 방에 대한 세션이 존재하면 true, 아니면 false

    • getName

    getName(): String

    해당 함수를 호출한 스크립트의 이름을 반환합니다.



    리턴 값

  • 해당 함수를 호출한 스크립트의 이름

    • setPower

    setPower(power: Boolean): void

    스크립트의 활성화 여부를 설정합니다.



    파라미터

  • power : 스크립트의 활성화 여부를 설정할 불린 값 (true: 활성화, false: 비활성화)

    • getPower

    getPower(): Boolean

    스크립트의 활성화 여부를 반환합니다.



    리턴 값

  • 스크립트가 활성화 상태이면 true, 아니면 false

    • compile

    compile(): void

    스크립트를 컴파일합니다.

    unload

    unload(): void

    스크립트를 언로드(컴파일 전 상태로 복구)합니다.

    on

    on(eventName: String, listener: Function): void

    이벤트 리스너를 등록합니다. (이벤트 리스너를 리스너 리스트의 맨 마지막에 추가)



    파라미터

  • eventName : 대상 이벤트
  • listener : eventName 이벤트 발생 시 호출될 함수(이벤트 리스너)

    • addListener

    addListener(eventName: String, listener: Function): void

    on()과 동일합니다.



    파라미터

  • eventName : 대상 이벤트
  • listener : eventName 이벤트 발생 시 호출될 함수(이벤트 리스너)

    • off

    off(eventName: String, listener?: Function): void

    마지막에 등록되거나 지정한 이벤트 리스너를 해제합니다.



    파라미터

  • eventName : 대상 이벤트
  • listener? : eventName 이벤트에서 해제할 이벤트 리스너

    • removeListener

    removeListener(eventName: String, listener?: Function): void

    off()와 동일합니다.



    파라미터

  • eventName : 대상 이벤트
  • listener? : eventName 이벤트에서 해제할 이벤트 리스너

    • removeAllListeners

    removeAllListeners(eventName: String): void

    지정된 이벤트에 등록된 모든 이벤트 리스너를 해제합니다.



    파라미터

  • eventName : 모든 이벤트 리스너를 해제할 대상 이벤트

    • prependListener

    prependListener(eventName: String, listener: Function): void

    이벤트 리스너를 등록합니다. (이벤트 리스너를 리스너 리스트의 맨 앞에 추가)



    파라미터

  • eventName : 대상 이벤트
  • listener : eventName 이벤트 발생 시 호출될 함수(이벤트 리스너)

    • listeners

    listeners(eventName: String): Array<Function>

    지정된 이벤트에 등록된 이벤트 리스너들을 배열로 반환합니다.



    파라미터

  • eventName : 등록된 이벤트 리스너들의 대상 이벤트


    • 리턴 값

  • eventName 이벤트에 등록된 이벤트 리스너들의 배열

    • markAsRead

    markAsRead(room?: String, packageName?: String): Boolean

    메시지가 수신된 방 또는 지정된 방의 메시지를 읽음 처리합니다.



    파라미터

  • room? : 알림을 읽을 채팅방
  • packageName? : 알림을 읽을 메신저 앱의 패키지명


    • 리턴 값

  • 알림 읽기 액션을 찾을 수 있으면 true, 아니면 false
