getCurrentBot

getCurrentBot(): Bot

현재 스크립트에 할당된 Bot을 반환합니다.



리턴 값

  • 스크립트에 할당된 Bot객체

    • getBot

    getBot(botName: String): Bot

    특정한 Bot을 반환합니다.



    파라미터

  • botName : 반환받을 Bot의 이름


    • 리턴 값

  • botName 이란 이름을 가진 Bot

    • getRooms

    getRooms(packageName?: String): Array<String>

    모든 메신저 앱 또는 특정 메신저 앱에서 받은 메시지들의 방 이름을 배열로 반환합니다.



    파라미터

  • packageName? : 메시지를 보낸 방들의 메신저 앱


    • 리턴 값

  • (packageName? 의 값이 없는 경우) 모든 메신저 앱에서 받은 메시지들의 방 이름의 배열
  • (packageName? 의 값이 있는 경우) packageName? 앱에서 받은 메시지들의 방 이름의 배열

    • getBotList

    getBotList(): ArrayArray<Bot>

    모든 Bot인스턴스를 배열로 반환합니다.



    리턴 값

  • 모든 Bot인스턴스의 배열

    • getPower

    getPower(botName: String): Boolean

    특정 Bot의 활성화 여부를 반환합니다.



    파라미터

  • botName : 활성화 여부를 확인할 Bot의 이름


    • 리턴 값

  • botName 이란 이름을 가진 Bot이 활성화 상태이면 true, 아니면 false

    • setPower

    setPower(botName: String, power: Boolean): void

    특정 Bot의 활성화 여부를 설정합니다.



    파라미터

  • botName : 활성화 여부를 설정할 Bot의 이름
  • power : 스크립트의 활성화 여부를 설정할 불린 값 (true: 활성화, false: 비활성화)

    • compile

    compile(botName: String, throwOnError?: Boolean): Boolean

    특정 Bot을 컴파일합니다.



    파라미터

  • botName : 컴파일할 Bot의 이름
  • throwOnError? : 컴파일 에러를 throw할 것인지의 여부 (기본값: false)


    • 리턴 값

  • botName 이란 이름을 가진 Bot의 컴파일을 성공하면 true, 아니면 false

    • compileAll

    compileAll(): void

    모든 Bot을 컴파일합니다.

    prepare

    prepare(botName: String, throwOnError?: Boolean): Int

    특정 Bot을 한 번도 컴파일한 적이 없을 경우 컴파일합니다.

    파라미터

  • botName : 한 번도 컴파일한 적이 없는 경우 컴파일할 Bot의 이름
  • throwOnError? : 컴파일 에러를 throw할 것인지의 여부 (기본값: false)


    • 리턴 값

  • 컴파일을 실패했을 때: 0
  • 컴파일을 성공했을 때: 1
  • 1번 이상 컴파일 되었을 때: 2

    • prepareAll

    prepareAll(throwOnError?: Boolean): Int

    모든 Bot에 대하여 prepare()를 실행합니다.



    파라미터

  • throwOnError? : 컴파일 에러를 throw할 것인지의 여부 (기본값: false)


    • 리턴 값

  • 새롭게 컴파일된 Bot의 개수

    • isCompiled

    isCompiled(botName: String): Boolean

    특정 Bot의 컴파일 완료 여부를 반환합니다.



    파라미터

  • botName : 컴파일 완료 여부를 확인할 Bot의 이름


    • 리턴 값

  • botName 이란 이름을 가진 Bot의 컴파일이 완료된 상태이면 true, 아니면 false

    • unload

    unload(botName: String): void

    특정 Bot을 언로드(컴파일 전 상태로 복구)합니다.



    파라미터

  • botName : 언로드할 Bot의 이름
