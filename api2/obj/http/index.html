request

            request(url: String, callBack: Function): void
            request(option: Object, callBack: Function): void

특정 URL에 Jsoup을 통해 비동기적으로 request를 보냅니다.



파라미터

  • url : request를 보낼 URL
  • option : request 옵션 (옵션 구조)
  • callBack : request 후 실행할 콜백 함수 (인자 정보)

    • requestSync

                requestSync(url: String): org.jsoup.nodes.Document
            requestSync(option: Object): org.jsoup.nodes.Document

    특정 URL에 Jsoup을 통해 동기적으로 request를 보냅니다.



    파라미터

  • url : request를 보낼 URL
  • option : request 옵션 (옵션 구조)

    • CallBack 인자 정보

                error: java.lang.Exception
            response: org.jsoup.Connection.Response
            doc: org.jsoup.nodes.Document

    request 옵션 구조

                {
             "url" : String
             "timeout" : Number
             "method" : String
             "headers" : { /* Header */ }
            }
  • url : request를 보낼 URL
  • timeout : request 타임아웃 (ms, 기본값: 3000)
  • method : request 메소드 (기본값: "GET")
  • headers : request 헤더 (기본값: null)
