exists

exists(fileName: String): Boolean

스크립트 로컬 폴더 내 Database폴더에 특정 파일이 존재하는지의 여부를 반환합니다.



파라미터

  • fileName : 존재 여부를 확인할 파일 이름


    • 리턴 값

  • fileName 파일이 존재하면 true, 아니면 false

    • readObject

    readObject(fileName: String): Object

    스크립트 로컬 폴더 내 Database폴더의 특정 파일을 읽어 자바스크립트 객체로 변환한 값을 반환합니다. (파일 내용이 JSON형태여야 합니다.)



    파라미터

  • fileName : 자바스크립트 객체형태로 읽을 파일 이름


    • 리턴 값

  • fileName 파일을 자바스크립트 객체로 변환한 값

    • readString

    readString(fileName: String): String

    스크립트 로컬 폴더 내 Database폴더의 특정 파일을 읽은 값을 반환합니다.



    파라미터

  • fileName : 읽을 파일 이름


    • 리턴 값

  • fileName 파일의 내용

    • writeObject

    writeObject(fileName: String, obj: Object): void

    스크립트 로컬 폴더 내 Database폴더 내 파일의 내용을 자바스크립트 객체를 문자열로 변환한 값으로 하여 생성 또는 수정합니다.



    파라미터

  • fileName : 생성 또는 수정할 파일의 이름
  • obj : 문자열화 하여 fileName 파일의 내용으로 정할 자바스크립트 객체

    • writeString

    writeString(fileName: String, str: String): void

    스크립트 로컬 폴더 내 Database폴더 내 파일의 내용을 특정 문자열 값으로 하여 생성 또는 수정합니다.



    파라미터

  • fileName : 생성 또는 수정할 파일의 이름
  • str : fileName 파일의 내용으로 정할 문자열 값
